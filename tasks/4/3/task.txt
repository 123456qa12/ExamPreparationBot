По каналу связи передаются сообщения, содержащие только четыре буквы: Д, Е, Ж, З. Для передачи используется двоичный код, допускающий однозначное декодирование.
Для букв Д, Ж, З используются кодовые слова:

Д — 11

Ж — 00

З — 010

Укажите кратчайшее кодовое слово для буквы Е, при котором код будет допускать однозначное декодирование. Если таких кодов несколько, укажите код с наименьшим числовым значением.
